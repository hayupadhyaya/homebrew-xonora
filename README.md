# homebrew-xonora

Homebrew tap for [Xonora](https://github.com/hayupadhyaya/xonora-cli) tools.

## Install

```sh
brew tap hayupadhyaya/xonora
brew install xonora-cli
```

## Available formulae

- **xonora-cli** — native C++ terminal client for Music Assistant
